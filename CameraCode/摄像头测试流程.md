  使用步骤

  第1步 — 先验证 UART 硬件（在 OpenART 上）：
  import camera_test
  camera_test.run(test_number=1)  # 短接 TX/RX，自发自收

  第2步 — MCU 联调（两端同时运行）：
  # OpenART 端 (CameraCode/camera_test.py):
  import camera_test
  camera_test.run(test_number=4)  # 先纯通信测试

  # RT1021 端 (将 (MCU)test_camera.py 复制到 CODE/ 目录):
  import test_camera
  test_camera.start()             # 启动通信，发心跳+收检测帧

  第3步 — 全功能测试：
  # OpenART 端:
  import camera_test
  camera_test.run(test_number=3, duration=60)  # 60 秒全功能

  # RT1021 端:
  import test_camera
  test_camera.start()

  协议帧格式

  [0xAA] [CMD:1B] [LEN:1B] [DATA:N bytes] [CRC:1B]
    ↑       ↑        ↑           ↑            ↑
   帧头    命令    数据长度    转义后数据    XOR校验

  ┌──────┬──────────┬─────────────────────────────────┐
  │ CMD  │   含义   │              Data               │
  ├──────┼──────────┼─────────────────────────────────┤
  │ 0x0F │ 心跳     │ 无                              │
  ├──────┼──────────┼─────────────────────────────────┤
  │ 0x10 │ 检测结果 │ [个数][cls,score,x1,y1,x2,y2]×N │
  ├──────┼──────────┼─────────────────────────────────┤
  │ 0x20 │ 文本消息 │ UTF-8 字符串                    │
  ├──────┼──────────┼─────────────────────────────────┤
  │ 0xF0 │ 回环测试 │ 任意数据（原样返回）            │
  └──────┴──────────┴─────────────────────────────────┘

  已知限制

  - OpenART / RT1021 MicroPython 固件的 UART 内部缓冲区约 63-64 字节
  - 单次 uart.write() 超过 63 字节会被截断（固件层限制，非硬件故障）
  - camera_test.py 已内置分块写入 (write_chunked) 绕开此限制
  - send_frame() 和大型回环测试均自动使用分块模式
